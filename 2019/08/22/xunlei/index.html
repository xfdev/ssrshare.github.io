迅雷极速版——绿色优化版发布，满速破解下载迅雷

简介

此软件是基于官方迅雷极速版ThunderSpeed 1.0.35.366 经过精简优化制作而成，集成Thunder L3、L4，
此安装程序为绿色自解压程序，可解压到桌面或系统磁盘根目录或非系统磁盘任意目录（默认为d:\Program Files\Thunder），
点击桌面图标“迅雷极速版(集成Thunder L3)或迅雷极速版(集成Thunder L4)或迅雷极速版”运行程序即可。

使用方法

迅雷极速版ThunderSpeed 1.0.35.366 绿色优化版【集成Thunder L3、L4】特点

  1. 去除升级组件、后台服务组件、游戏组件等多余组件…

  2. 集成账号防踢补丁（Windows 10系统千万不要开启本项，开启会闪退，不开启运行正常稳定）

  3. 支持自定义边下边播的播放器（需要手动在 Xmp.ini 设置自定义播放器的路径）

  4. 禁止Data目录ThunderPush自动下载推送垃圾文件

  5. 添加迅雷下载专用链支持（支持多浏览器）

  6. 集成Thunder L3、Thunder L4，可根据需求选择使用

  7. 可根据需要使用本地VIP会员补丁

  8. Windows 10系统也能正常使用

敏感资源下载方法【不保证用这种方法能解决所有下载问题】

1.打开【计算机】，找到目录

C:\Windows\System32\drivers\etc

2.找到文件hosts，右键使用记事本打开（如果找不到此文件请使用自带的文件搜索功能）

3.在最下面一行添加以下代码：

127.0.0.1 hub5emu.sandai.net
127.0.0.1 hub5btmain.sandai.net
127.0.0.1 upgrade.xl9.xunlei.com

//使用下面这项会造成emule无法下载

127.0.0.1 hub5emu.sandai.net

//使用下面此项可能会造成BT下载缓慢

127.0.0.1 hub5btmain.sandai.net
