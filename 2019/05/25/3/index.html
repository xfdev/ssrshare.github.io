偷电被抓——比特币“挖矿”被发现 辽宁绥中抓获“电耗子”

发表于 2019-05-25 | 分类于 新闻

比特币“挖矿”疯狂窃电 辽宁绥中抓获“电耗子”

辽宁省绥中县公安局近日侦破一起为408台比特币“挖矿”机供电偷电团伙。由于设备耗电量巨大，为了降低成本，采取偷电方式来维持设备运转，这伙“电耗子”85天内疯狂窃电价值90余万元。目前，408台比特币“挖矿”机、100多台备用“挖矿”机全部被扣押，3名涉案人员已被刑拘。

　　绥中县公安局西甸子派出所所长蒋红新告诉记者，绥中县西甸子供电线路由万家66千伏变电站供电，这条线路同期系统日损失率2018年均为6%以下，2019年以来，电力损耗异常飙升，最高达到32.58%。绥中县电力部门多次组织相关人员分析造成线路高损的原因，没有找到问题症结，随后向派出所报案。

　　接到报案后，蒋红新立即指派民警展开侦查工作。走访摸排中民警了解到，附近村子有一户平房出租给一外地租房户，一般情况下，很少有外地人来此租房，且有村民反映，平房内经常传出很大的风扇轰鸣声，这引起了侦查民警的警觉。在接下来的走访中，民警还了解到，这户人家很少开门，村民都没怎么见过他们，且每天都有人开车在周边“放哨”，这更引起了民警的怀疑。经进一步侦查，警方确定该住户有重大窃电嫌疑，将2名嫌疑人抓获。

　　“我们在现场看到，平房内全是比特币‘挖矿’机，还在不停地运转。”蒋红新说。

　　经办案民警查明，2人受雇于一个叫郑某的老板，这些窃电设备都是从北京和南京快递过来的，于去年11月安装。为防止老板郑某事发逃窜，办案民警及时封锁消息并将其抓捕归案。

　　郑某交代，他们租借偏僻农村民房，私改电线，窃电用于400多台“挖矿”机，挖取比特币牟利。

　　经警方和供电部门核查，犯罪嫌疑人郑某及其同伙85天时间窃电价值达90余万元。目前，3人已被刑拘。案件在进一步侦办中。
