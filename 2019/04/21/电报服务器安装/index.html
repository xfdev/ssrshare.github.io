telegram代理服务——服务器一键配置Telegram代理

发表于 2019-04-21 | 更新于 2019-07-20 | 分类于 服务器脚本

telegram代理使用更加方便，不用其他梯子

安装和配置

一键安装源

  • 请输入 MTProxy 端口 1-65535(默认: 7000)
  • 请输入 MTProxy 密匙（手动输入必须为32位，[0-9][a-z][A-Z]，建议随机生成）
  • 请输入 MTProxy 的 TAG标签（TAG标签只有在通过官方机器人 @MTProxybot 分享代理账号后才会获得，不清楚请留空回车）
  • 如果本机是NAT服务器（谷歌云、微软云、阿里云等），则请输入你的服务器内网IP，否则会导致无法使用。如果不是请直接回车！
  • 然后把https开头的连接分享给别人就可以了（选择你设定端口的连接）

官方分享 可以添加自己的推广频道

官方的这种机制非常好，既可以分享又可以推广自己

  • 添加机器人 https://t.me/MTProxybot 或 @MTProxybot
  • /newproxy - 添加一个新的代理 按要求输入 ip：端口和密钥
  • 然后机器人会告诉你一个唯一的TAG标签,这时候你去运行脚本 - 7.设置 账号配置 - 3.修改 TAG 配置 -填写TAG标签，然后脚本会自动重启代理服务端。
  • /myproxies - 查看已分享的代理账号 点击Set promotion - 配置推广频道
  • 接下来你输入想要推广的频道就可以了,机器人就会让你说出要推广的频道链接，可以是 t.me/xxx 格式，也可以是 @xxx 格式
